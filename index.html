What is Havoc Preset?

Havoc Preset is a Roblox key system UI library. Loaded externally via loadstring and configured entirely from your own script. Once a valid key is redeemed (or auto-loaded from file), getgenv().SCRIPT_KEY is set and your game script continues. The library also returns itself as Hub.

Tip — The library is CDN-hosted. You write a loader with Junkie_Configs and UI_Configs, merge them into Config, and call Hub.new(Config):Init(). Use the Code Generator tab to have your config written for you automatically.
02 — Flow
How It Works
1
Load the Library
Your script calls loadstring(game:HttpGet(LIBRARY_URL))() inside a pcall to fetch the Hub library from CDN.
2
Auto-Login Check
TryAutoLogin() runs first — it checks for a saved key in KeyFile or getgenv().SCRIPT_KEY and validates via Junkie SDK. If valid, the UI is skipped entirely.
3
Show UI if Needed
If auto-login fails, Hub.new(Config) and hub:Init() render the key system UI for the user to enter their key.
4
Wait for Key
while not getgenv().SCRIPT_KEY do task.wait(0.1) end blocks until a valid key is entered. Your game script runs below that line.
03 — Template
Minimal Template
Lua — Minimal Template
getgenv().SCRIPT_KEY = nil

local Junkie_Configs = { }
local UI_Configs     = { }
local Config = { }

local Hub
pcall(function()
  local src = game:HttpGet("LIBRARY_URL")
  if #src < 100 then return end
  Hub = loadstring(src)()
end)
if type(Hub) ~= "table" then return end

local autoLogin = TryAutoLogin()
if not autoLogin then
  pcall(function()
    local hub = Hub.new(Config)
    hub:Init()
  end)
end

while not getgenv().SCRIPT_KEY do
  task.wait(0.1)
end
04 — Core
Junkie_Configs
Controls Junkie SDK integration — key validation, webhook relay, and provider setup.
PropertyTypeDescription
UseJunkiebooleantrue loads the Junkie SDK for real-time key validation. false accepts any key (testing mode).
Junkie_WebhookstringYour Junkie relay webhook URL. Notifies you on key redemption. Leave "" to disable.
ServicestringYour service name from the Junkie dashboard. Must match exactly.
IdentifierstringYour user/dashboard identifier from Junkie.
ProviderstringThe provider name configured in your Junkie dashboard.
MaxKeyAttemptsnumberFailed attempts before UI force-closes. Recommended: 5.
Premiumbooleantrue loads the Premium script URL from GameSupport instead of Free.
Providers
Providers
Array of key provider buttons shown in the UI. Each can be shown, hidden, or pinned.
PropertyTypeDescription
NamestringDisplay name on the provider button.
Iconstringrbxassetid for the icon. Use "rbxassetid://0" for the built-in default.
UrlstringURL opened on click. Fallback if get_key_link() returns nil.
Showbooleantrue = visible in the UI.
IsJunkiebooleanExactly ONE provider should have this. Enables SDK's get_key_link() flow.
Pinnedbooleantrue = rendered first with a highlight border.
Warning — Only one provider should have IsJunkie = true. If none, all providers render as regular URL buttons.
GameSupport
GameSupport
Maps Roblox Place IDs to script URLs. After key acceptance the matching script loads. Universal is required as a fallback.
Lua
GameSupport = {
  [2788229376] = {
    Name    = "Game Name Here",
    Free    = "https://api.jnkie.com/.../HASH/download",
    Premium = "https://api.jnkie.com/.../HASH/download",
  },
  Universal = {
    Name    = "Universal",
    Free    = "https://api.jnkie.com/.../HASH/download",
    Premium = "https://api.jnkie.com/.../HASH/download",
  },
}
Warning — Replace all HASH placeholders with your actual Junkie script hashes. The Universal entry is required as a fallback.
05 — Visual
UI_Configs
Controls the visual appearance and behaviour of the key system UI.
Navigate using the sidebar or Next button to reach General, Background, Snow, Key System, Discord, Socials, Instructions, and Changelog sub-sections. Or use the Code Generator tab to configure everything visually.
General
General Settings
PropertyTypeDescription
NamestringHub display name in the title section.
VersionstringVersion pill under the hub name. e.g. "1.0"
ColorColor3Primary accent colour — buttons, borders, glows, badges.
Text_ColorColor3/""Text on coloured buttons. Leave "" to auto-pick black/white.
Iconstringrbxassetid for your hub logo in the title bar.
BadgeTextstringBadge label next to the icon. e.g. "KEY SYSTEM"
CustomFontstringrbxassetid of a custom font. Leave "" for built-in Gotham.
ShowOutlinebooleantrue = accent-coloured border around the main panel.
Background & Overlay
Background & Overlay
PropertyTypeDescription
GradientAnimatedbooleantrue = background gradient slowly rotates.
GradientAnimationSpeednumberSeconds per full rotation. Lower = faster.
GlassOpacitynumberPanel transparency. 0 = opaque, 1 = invisible. 0.18 = frosted glass.
SlideInEnabledbooleantrue = panel slides up on open.
BlurEnabledbooleantrue = applies BlurEffect to Lighting while UI is open.
BlurSizenumberBlur strength 0–56.
OverlayOpacitynumberBackground overlay darkness. 0 = none, 1 = black.
Snow
Snow Settings
PropertyTypeDescription
SnowEnabledbooleantrue = falling snowflake particles over the background.
SnowDensitynumberSnowflake count multiplier. 1.0 = default.
SnowColorColor3Snowflake colour.
SnowSize{Min,Max}Pixel diameter range. e.g. {Min=3, Max=7}
Key System
Key System Settings
PropertyTypeDescription
KeySystemLabelstringLabel text above the key input box.
KeySystemPlaceholderstringAnimated placeholder that types itself when the box is empty.
KeyFilestringFilename used to save and reload a verified key. Leave "" to disable.
CloseOnSuccessbooleantrue = UI auto-closes after successful redeem.
SuccessMessagestringToast shown on success. e.g. "✓ Access Granted!"
Auto-Login — On launch the library checks KeyFile against the Junkie SDK. If valid, the UI is never shown.
KEYLESS mode — If Junkie returns message == "KEYLESS", the library sets getgenv().SCRIPT_KEY = "KEYLESS" and proceeds without saving any file.
Discord
Discord Settings
PropertyTypeDescription
DiscordstringYour invite code (part after discord.gg/). Powers the Join button and live member count.
ShowMemberCountbooleantrue = fetches and shows live member + online count.
Socials & AppLinks
Socials & AppLinks
Both arrays share the same field structure.
PropertyTypeDescription
NamestringDisplay name. Used to auto-resolve icon if Icon is "rbxassetid://0".

LinkstringURL copied to clipboard on click.
Showbooleantrue = visible.
Pinnedbooleantrue = rendered first with a highlight border.
Auto-icons — Set Icon to "rbxassetid://0" for built-in icons: Discord, Twitter, YouTube, TikTok, Instagram, Telegram, Reddit, GitHub, Guilded, Patreon, Ko-fi, Twitch, V3rm, Website, Linkvertise, LootLabs, Work.ink, Shrtfly, Lockr, Cuty, ShrinkEarn, Rinku.

07 — Example
Full Config Example
A complete, working config you can copy and customise.
Lua — Full Config
getgenv().SCRIPT_KEY = nil

local Junkie_Configs = {
  UseJunkie       = true,
  Junkie_Webhook  = "https://jnkie.com/api/v1/webhooks/relay/YOUR_ID/send",
  Service         = "My Hub",
  Identifier      = "1234",
  Provider        = "My Hub",
  MaxKeyAttempts  = 5,
  Premium         = false,
  Providers = {
    { Name="Junkie Development", Url="https://jnkie.com/junkiehub", Show=true, IsJunkie=true, Pinned=true },
    { Name="Linkvertise", Url="https://linkvertise.com/yourhub", Show=true, Pinned=false },
  },
  GameSupport = {
    [2788229376] = { Name="Blox Fruits", Free="...", Premium="..." },
    Universal    = { Name="Universal",   Free="...", Premium="..." },
  },
}

local UI_Configs = {
  Name = "My Hub", Version = "1.0",
  Color = Color3.fromRGB(138, 43, 226),
  GradientAnimated = true, GradientAnimationSpeed = 4,
  BadgeText = "KEY SYSTEM", ShowOutline = true,
  KeySystemLabel       = "Enter Key",
  KeySystemPlaceholder = "jnk-XXXXXXXXXXXXXXXX",
  ShowMemberCount = true,
  SnowEnabled = true, SnowDensity = 1.0,
  GlassOpacity = 0.18, SlideInEnabled = true,
  BlurEnabled = true, BlurSize = 30, OverlayOpacity = 0.65,
  CloseOnSuccess = true, SuccessMessage = "✓ Access Granted!",
  KeyFile = "myhub_key.txt", Discord = "your_invite_code",
}

local Hub
pcall(function()
  local src = game:HttpGet("LIBRARY_URL")
  if #src < 100 then return end
  Hub = loadstring(src)()
end)
if type(Hub) ~= "table" then return end

while not getgenv().SCRIPT_KEY do task.wait(0.1) end

return Hub
